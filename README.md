## Готовая сборка на основе NEXT 14

## Старт

Для запуска сервера разработки:

```bash
npm run dev
```

## Структура проекта

```bash
api => функции запросов на серверр
app => роутинг
assets => 
  fonts => шрифты
  images => статичные картинки которые используются на страницах, для каждой страницы своя папка 
  icons => svg, png, jpg иконки
components = >
  global => компоненты которые используются на всех страницах, напримемер <Container/>, <Layout/>, <Header/> и т.п.
  icons => svg иконки в виде реакт компонентов
  pages => составные компоненты для страниц, для каждой странцы своя папка
  ui => мелкие переиспользуемые компоненты которые имеют какой то функционал, напрмер <Button/>, <Checkbox/>, <Select/> и т.п.
  widgets => переиспользуемые блоки или большие и средние компоненты с каким то функционалом, которые можно модключить на любой странице
css => 
  common => переменные 
  global => глобальные стили и дополнительные стили библеотек  
hooks => кастомные хуки
utils => дополнительные функции и хелперы  
types => все типы проекта
```
